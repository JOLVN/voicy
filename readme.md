# VOICY

---

## Stack

### API
* Nest JS
* TypeORM

### APPP
* Sveltekit
* Vite JS

## Run the API

* Install dependencies
```npm install```

* Create a .env file similar to .env-example and fill it with your data

* Go to the api folder
```cd api```

* Run docker compose
```make up```

* Run server
```npm run start```


## Run the APP

* Install dependencies
```npm install```

* Start app
```npm run dev```
